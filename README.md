## Description

BlogPost Genie is a blog site where developers can publish their blog posts and comment on other developersâ€™ posts as well. This application follows the MVC paradigm in its architectural structure, uses Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication.

Created by [Nate Greeff](https://github.com/NateGreeff)

## Features

- **User Authentication**: Secure sign up and log in functionality.
- **Homepage**: Displays all existing blog posts.
- **Navigation Links**: Easy access to homepage, blogposts, and login/signup.
- **Blog Management**: Add blog posts.
- **Comments**: Readers can leave comments on blog posts.
